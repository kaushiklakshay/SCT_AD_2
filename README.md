# Android To-Do List App
A sleek, efficient task management application built for Android. Designed to help users organize their daily routines, track goals, and stay productive with a clean, distraction-free user interface.
### 🚀 Key Features
* Add tasks instantly with a simple input bar.
* Check off completed tasks with a single tap.
* Delete tasks when they are no longer needed.
* Lightweight and fast startup.

### 🛠️ Built With
* **Language:** Kotlin/Java
* **UI Components:** RecyclerView, CardView, FloatingActionButton
* **IDE:** Android Studio
## 📲 How to Install

1. Navigate to the [Releases] section of this repository.
2. Download the latest `.apk` file.
3. Open the file on your Android device and install it (ensure "Install from Unknown Sources" is enabled).
